Credit Card Approvals

The credit score of a person is their creditworthiness based on their credit history. The higher this number, the more financially trustworthy a person is.

While building this credit card predictor, the data had to be cleaned to build the machine learning algorithm. Some of the most widely-known feature engineering steps such as scaling, label encoding, and missing value imputation had to be done to make the data tidy. The data was trained to run in a logistic regression algorithm to predict if a person's application for a credit card would get approved or not by giving some information about that person. The model performed well by acquiring an accuracy of 85%.
